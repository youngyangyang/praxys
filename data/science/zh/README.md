# Science YAML — Chinese translations

This directory mirrors the structure of `data/science/` and stores Simplified
Chinese translations of theory + label files. The loader in
`analysis/science.py` prefers a localized file here and silently falls back to
the English source in the parent directory when a translation is missing, so
partial coverage is safe to ship.

Files in this tree are generated by
[`scripts/translate_missing.py yaml`](../../scripts/translate_missing.py) via
the `.github/workflows/i18n.yml` workflow. Review the PR diff before merging
— technical terms (CP, HRV, TSB, RSS) should stay in English, and markdown
tables should keep their structure.
